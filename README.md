# laofi-app

Aplicación web para coworking laofi.

## Funcionalidades principales

- Comprar café desde el celular
- Pago con Mercado Pago, MODO o efectivo
- Notificación automática de pedidos a la recepción

## Tecnologías

- Next.js (TypeScript, App Router)
- Tailwind CSS
- Node.js/Express (backend para pedidos y pagos)

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```
